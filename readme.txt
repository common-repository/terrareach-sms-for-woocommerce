SMS gateway for Sri Lanka to send transactional or bulk SMS to your customers via WooCommerse store.

== Description ==

TerraReach is a popular SMS gateway for Sri Lanka which you can use to send transactional or bulk SMS to your customers. This plugin allows you to integrate TerraReach SMS service into your WooCommerce store.

**Note:** This plugin requires an active TerraReach account and API key to function. It communicates with the TerraReach API to send SMS messages. By using this plugin, you agree to the TerraReach [Terms of Service](https://docs.terrareach.com/legal/terms-of-service) and [Privacy Policy](https://docs.terrareach.com/legal/privacy-policy).

= Features =

* Send SMS to your customers automatically for any WooCommerce order status. You can let them know if you received the order, when they make the payment, you process the order, when the order is ready, when order is shipped or any custom event you define.
* Receive an SMS to one or multiple admin mobile numbers when your store got a new order.
* Send SMS when you add a customer note.
* You can customize the content of every automated SMS. 

== Installation ==

= Automatic Installation =

1. In the dashboard navigate to `Plugins > Add New`
2. Search for `TerraReach SMS for WooCommerce`
3. Locate the `TerraReach SMS for WooCommerce` by `Origyn` and click `Install Now`
4. After installation, click on `Activate Plugin` to activate

= Manual Installation =

1. Download the zip file using the `Download` button above in this page
2. In the dashboard navigate to `Plugins > Add New`
3. Click on `Upload Plugin`
4. Click `Choose File` then select the zip file you downloaded
5. Click `Install Now`, wait for the installation and click `Activate Plugin`

= Enabling the Landing Page =

1. In the dashboard, navigate to `WooCommerce > Settings`
2. Go to `TerraReach SMS` tab
3. Fill all options required, and save the settings

== Frequently Asked Questions ==

= How to sign-up for an account? =
Go to our website https://www.terrareach.com and register for an account.

= Can this plugin send SMS? =
Yes, if you are registered TerraReach user, you can integrate this plugin with TerraReach and trigger SMS automatically for WooCommerce order status updates and order note updates as well.
